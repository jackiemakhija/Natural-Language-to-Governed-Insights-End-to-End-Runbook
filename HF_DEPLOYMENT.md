# Hugging Face App & GitHub Repository

## Hugging Face Space

You can try the app live on Hugging Face Spaces:

[Natural-Language-to-Governed-Insights on Hugging Face Spaces](https://huggingface.co/spaces/jackiemakhija/Natural-Language-to-Governed-Insights)

### Usage

1. Visit the Hugging Face Space above.
2. Enter your natural language query in the input box.
3. Click the submit/run button to generate governed insights.

### Deploying to Hugging Face Spaces

To deploy this app to your own Hugging Face Space:

1. Fork or clone this repository (see GitHub URL below).
2. Create a new Space on Hugging Face (select Gradio or Streamlit as appropriate).
3. Push your code to the Hugging Face Space repository.
4. Ensure your `requirements.txt` is present for dependency installation.
5. The app will build and launch automatically.

For more details, see the [Hugging Face Spaces documentation](https://huggingface.co/docs/hub/spaces).
